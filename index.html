Welcome to Central Perf Project.

This project was made to help users to manage performance testing session from a central server. Web UI will offer to organize your projects/run/scripts.

Performance testing is done by Jmeter and/or Gattling and real time charts are generated during test.

Results and comments are stored in runs history.

Central Perf is still in construction (first beta coming soon) and will be distributed under AGPL 3 licence.
